# CQU-TAL: Chongqing University Traffic Accident Law Dataset

## 简介

**CQU-TAL** 是一个面向交通法规与交通事故智能理解的中文数据集，由重庆大学研究团队构建。该数据集将真实交通事故案例与道路交通法规条款进行深度关联，旨在为智能交通执法、法律智能辅助、事故责任判定及自动驾驶伦理决策等研究提供高质量的数据支持。

## 数据集特点

- **法规与事故联动**：每条事故样本不仅包含事故描述、车辆信息、环境要素，还标注了所涉及的交通法规条款及其责任判定依据。
- **结构化标注**：事故文本经自然语言处理与人工校对，抽取了关键实体（如参与者、行为、信号灯状态）与事件关系，形成结构化的场景表示。
- **多源融合**：数据来源于公开交通事故报告、交警部门脱敏案例以及模拟场景，覆盖城市道路、高速公路等多种环境。
- **可扩展性**：数据集以 JSON/CSV 格式提供，方便研究者直接使用或二次开发。

## 数据规模与内容

- **事故样本数**：初期版本包含 10,000+ 条真实事故记录（持续扩充中）。
- **法规条款库**：收录《道路交通安全法》及其实施条例的核心条款，并进行条款拆分与规则化。
- **标注字段**：
  - 事故描述（原始文本）
  - 时间、地点、天气、路况
  - 涉事方信息（车辆类型、驾驶员状态、行为序列）
  - 事故原因（如未保持安全距离、闯红灯、酒驾等）
  - 责任认定（全部/主要/同等/次要责任）及依据的法规条款
  - 法律建议（参考处罚或赔偿责任）

## 构建流程

1. **数据采集**：爬取公开事故报告，与交管部门合作获取脱敏数据。
2. **法规结构化**：将法规文本拆解为条件-行为-后果三元组，构建法规知识图谱。
3. **事故解析**：使用 NLP 技术抽取事故要素，结合人工修正生成结构化表示。
4. **法规匹配**：通过规则引擎与人工审核，建立事故与法规条款的关联。
5. **质量校验**：由交通法律专家对样本进行抽样复核，确保标注准确性。

## 应用场景

- **智能法律咨询**：训练模型自动分析事故描述，提供法规参考与责任预判。
- **辅助交通执法**：为交警快速处理简易事故提供法规匹配建议。
- **驾驶培训系统**：基于真实案例生成教学材料，帮助学员理解交通法规。
- **自动驾驶伦理决策**：作为仿真场景中的法规约束基准，评估自动驾驶行为合法性。
